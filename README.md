This is an example of an automation testing project on the Android application API Demos 17 using Katalon Studio. In this project, there are two testing stages: functional and regression testing. The testing involves navigation through each menu, checkbox testing in the presentation feature, and SMS Messaging with valid and invalid numbers/names. The test cases for this project can be accessed through the following link: https://docs.google.com/spreadsheets/d/14vOZ_UCg1nWKyPR0PqNCxjq3uY37-59nk6QS3TmXEbo/edit?usp=sharing
